This interactive booklet is from Jo Flint's 2024 exhibition titled Re-Viewing the Gaze.

Please use the arrows to navigate through the pages.

The "Dark Mode" button enables dark mode for easier reading.

The "Menu" button offers other relevant links to the project and artist.

The "Overlay" button enables the project's interactive design, for more details please see page 1 of the booklet.

On mobile, rotate the screen to enable two page spread.

Press esc on computer or back on mobile to exit full screen and return to enter screen.

Please press the "i" at the top right to exit and return to this screen.
